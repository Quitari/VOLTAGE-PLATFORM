Доступные переменные

Нажмите на переменную чтобы скопировать. Используйте в тексте ответа бота.

{user} {viewers} {uptime} {random} {game} {title} {channel} {count}
{user}Никнейм того кто написал команду
{viewers}Текущее число зрителей
{uptime}Время с начала стрима
{random}Случайный зритель из чата
{game}Текущая игра на стриме
{count}Счётчик использований команды
